Jogos JavaScript

Jogos JavaScript é um projeto de desenvolvimento de jogos 2D utilizando javascript e html5. JavaScript é uma linguagem de script usada para proporcionar uma interação mais dinâmica entre o usuário e a página web, é suportada por praticamente todos os navegadores modernos, não sendo necessário a instalação de qualquer software ou plugin externo. A habilidade da linguagem renderizar gráficos em elementos canvas e capturar as ações do usuário faz dela uma opção interessante para o desenvolvimento de jogos web.

Aqui são apresentados alguns jogos que você poderá jogar sozinho ou com um amigo. O projeto está em fase inicial e novos jogos serão acrescentados futuramente.

Lista de jogos

  1. sinuquinha: Sinuquinha é um jogo de sinuca para um jogador. Acerte bolas na caçapa em sequência para garantir uma pontuação melhor. Veja quantos pontos você é capaz de marcar.
  2. memória movediça: Memória Movediça é um jogo da memória no qual de tempos em tempos algumas cartas sobre a mesa irão trocar de lugar. Teste sua memória e concentração neste jogo com nível de dificuldade aumentada.
